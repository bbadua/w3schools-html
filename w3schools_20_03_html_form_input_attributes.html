html5 input attributes

Attribute Example Code
autocomplete click 
<form action="/action_page.php" autocomplete="on">
  First name:<input type="text" name="fname">
  Last name: <input type="text" name="lname">
  E-mail: <input type="email" name="email" autocomplete="off">
  <input type="submit">
</form>

NOTE:
specifies whether a form or input field should have autocomplete on or off.

When autocomplete is on, the browser automatically completes the input values based on values that the 
user has entered before.

It is possible to have autocomplete "on" for the form, and "off" for specific input fields, or vice versa.

The autocomplete attribute works with <form> and the following <input> types: text, search, 
url, tel, email, password, datepickers, range, and color.
nonvalidate click 
<form action="/action_page.php" novalidate>
  E-mail: <input type="email" name="user_email">
  <input type="submit">
</form>

NOTE:
a <form> attribute.  When present, novalidate specifies that the
 form data should not be validated when submitted.
autofocus click 
<form action="/action_page.php">
  First name:<input type="text" name="fname" autofocus>
  Last name: <input type="text" name="lname">
  <input type="submit">
</form>

NOTE:
specifies that the input field should automatically get focus when the page loads.
form click 
<form action="/action_page.php" id="form1">
  First name: <input type="text" name="fname">
  <input type="submit" value="Submit">
</form>

...

Last name: <input type="text" name="lname" form="form1">

NOTE:
specifies one or more forms an <input> element belongs to.  To refer to more
 than one form, use a space-separated list of form ids.
formaction click 
<form action="/action_page.php">
  First name: <input type="text" name="fname">
  Last name: <input type="text" name="lname">
  <input type="submit" value="Submit">
  <input type="submit" formaction="/action_page2.php" value="Submit to another page">
</form>

NOTE:
specifies a form's action.  overrides the action attribute of the <form> element
formenctype click 
<form action="/action_page_binary.asp" method="post">
  First name: <input type="text" name="fname">
  <input type="submit" value="Submit">
  <input type="submit" formenctype="multipart/form-data" value="Submit as Multipart/form-data">
</form>

NOTE:
specifies how form data should be encoded when submitted
formmethod click 
<form action="/action_page.php" method="get">
  First name: <input type="text" name="fname">
  Last name: <input type="text" name="lname">
  <input type="submit" value="Submit">
  <input type="submit" formmethod="post" value="Submit using POST">
</form>

NOTE:
defines the HTTP method for sending form-data to the action URL.
overrides the method attribute of the <form> element.
attribute can be used with type=submit and type=image.
formnovalidate click 
<form action="/action_page.php">
  E-mail: <input type="email" name="userid">
  <input type="submit" value="Submit">
  <input type="submit" formnovalidate value="Submit without validation">
</form>

NOTE:
overrides the novalidate attribute of the <form> element.
attribute can be used with type="submit".
formtarget click 
<form action="/action_page.php">
  First name: <input type="text" name="fname">
  Last name: <input type="text" name="lname">
  <input type="submit" value="Submit as normal">
  <input type="submit" formtarget="_blank" value="Submit to a new window">
</form>

NOTE:
specifies a name or a keyword that indicates where to display the response that is received after submitting the form.
The formtarget attribute overrides the target attribute of the <form> element.
The formtarget attribute can be used with type="submit" and type="image".

  <input type="image" src="img_submit.gif" alt="Submit" width="48" height="48">
</form>

NOTE:
specify the height and width of an <input type="image"> element.

min and max click 
<form action="/action_page.php">

  Enter a date before 1980-01-01:
  <input type="date" name="bday" max="1979-12-31">

  Enter a date after 2000-01-01:
  <input type="date" name="bday" min="2000-01-02">

  Quantity (between 1 and 5):
  <input type="number" name="quantity" min="1" max="5">

  <input type="submit">

</form>

NOTE
specify the minimum and maximum values for an <input> element.
work with the following input types: number, range, date, datetime-local, month, time and week.
multiple click 
<input type="file" name="img" multiple>

NOTE:
specifies that the user is allowed to enter more than one value in the <input> element.
works with the following input types: email, and file.
pattern click 
<input type="text" name="country_code" pattern="[A-Za-z]{3}" title="Three letter country code">


specifies a regular expression that the <input> element's value is checked against.
works with the following input types: text, search, url, tel, email, and password.
placeholder click

  <input type="text" name="fname" placeholder="First name">
  <input type="text" name="lname" placeholder="Last name">
  <input type="submit" value="Submit">
</form>

NOTE:
specifies a hint that describes the expected value of an input field (a sample value or a short description of the format).
The hint is displayed in the input field before the user enters a value.
works with the following input types: text, search, url, tel, email, and password.

specifies that an input field must be filled out before submitting the form.
works with the following input types: text, search, url, tel, email, 
password, date pickers, number, checkbox, radio, and file.

specifies the legal number intervals for an <input> element.
