<div> vs <span>

div

hello
world
<div>hello</div>
<div>world</div>

NOTE: <div> is a block element.  It will start on a new line and take up an entire width.

span

hello world 
<span>hello</span>
<span>world</span>

NOTE: <span> is an inline element. Does NOT start a new line.  Does NOT take up entire width.  Can be place within other elements.
